# Images Directory

이 디렉토리에 사진 파일들을 저장하세요.

## 사용 방법

1. 이 폴더에 이미지 파일을 추가합니다 (예: `seoul-tower.jpg`)
2. `content/photos/` 폴더의 MD 파일에서 `/images/파일명.jpg` 형식으로 참조합니다

## 예시

이미지 파일 위치: `public/images/seoul-tower.jpg`
MD 파일에서 참조: `image: "/images/seoul-tower.jpg"`

## 권장 사항

- 파일명은 영문과 숫자, 하이픈(-)만 사용
- 적절한 크기로 최적화된 이미지 사용 (권장: 가로 1200-2000px)
- JPG 또는 PNG 형식 권장
